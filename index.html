Seasarサーバ構築コミュニティ

Seasarサーバ構築コミュニティとは

「Seasarサーバ構築コミュニティ」とは、Seasarプロジェクトで利用するサーバを構築･運営するコミュニティです。

Seasar.orgサーバについて

Seasar.orgサーバは現在次の状態にあります。

共通

  • 現在サーバは2台あり、1台を本サーバ、もう1台を実験サーバとしています。
    • 本サーバ: *.seasar.org
    • 実験サーバ: *.dev.seasar.org
      • 実験サーバは、予告無く仕様が変更される場合があります。
  • 実験サーバのサイトがあると紛らわしい場合は、本サイトに誘導するようなページに置き換えてください。使い方は各プロジェクトにお任せします。
  • プロジェクトごとにSVNリポジトリがあります。
    • wwwディレクトリにコミットしたものがプロジェクトサイトに自動反映されます。
    • コミットするには下記のツールが便利です。
      • Subclipse: Eclipse Pluginです。
      • TortoiseSVN: Windowsのエクスプローラに組み込んで利用出来ます。。
      • svn: UNIX, Cygwinなどでのコマンド

S2プロダクト

  • 各プロダクトのWEBコンテンツをホスティングしています。
  • ソースコードは、SourceForgeのCVSを利用しています。
  • メーリングリストは、SourceForgeのものを利用しています。

Sandbox

Seasar.orgサーバを利用していないプロダクト

  • 通常はこちらになり、OSCJのリソースを利用してます。
    • CVS
    • フォーラム
    • メーリングリスト

Seasar.orgサーバを利用しているプロダクト

  • 希望した一部のSandboxプロダクトはSeasar.orgサーバを利用しています。
  • サイトアドレスにsandboxが入ります。
    • 例: http://example.sandbox.seasar.org/
  • ソースコードにSeasar.orgサーバのSVNが利用出来ます。

アカウント申請方法

注意: Seasar.orgサーバのアカウントは、SourceForgeおよびOSCJのアカウントとは別物になります。

アカウントが必要なコミッタの方は、Seasar-oper メーリングリストに下記の情報を明記して申請してください。

  • 希望アカウント名 (第2候補までお願いします)
  • コミット権限を持つ所属プロダクト

アカウント発行後、発行したアカウント名と初期パスワードをSeasar-oper メーリングリストに登録してあるメールアドレスに送信します。

今後について

プロジェクトの運用方針について

現在明確なプロジェクトの運用方針が定まっていませんが、Seasar-oper メーリングリストで議論された結果に従い運用していきます。

機能について

現在のリソースが分散してしまっている状態を解決するために、Seasar.orgサーバのための専用WEBアプリケーションを開発していきます。最終的には、各コミッタ・メーリングリストのアカウントをすべて一括で管理出来るようになる予定です。
